# 【无忧存&小铁3】无忧存导流收取推广费功能说明文档

1.  **功能简介**
    
    在无忧存访问/预约了小铁寄存柜的用户，会被打上“无忧存导流”的标签，且有效时间为7天（再次在无忧存访问寄存柜会重新刷新为7天）
    
    带有标签的用户在寄存柜下单时，将自动扣除一定费用购买同金额的优惠券并使用（用户无感知）
    
    例如，预付费10元，该优惠券5元
    
    应付：预付费10元+优惠券5元-优惠券抵扣5元=10元。与预付费一致
    
    但订单详情中的实付金额是减去优惠券购买费用的金额
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/89695162-69a8-4003-8f40-d10c7860ad97.png)
    
2.  **后台管理系统操作指南**
    
    1.  **配置优惠券**
        
        配置优惠券路径： PC管理端 -> 【运营管理】 -> 【优惠券管理】
        
        点击“添加”在对应门店创建优惠券
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/148dfd5d-954f-4503-9cf9-72e11853eaff.png)
        
        优惠券信息要求（列出的信息为固定信息，所有此功能优惠券保持一致）：
        
        优惠券名称：【小铁无忧存专享】
        
        优惠券类型：无忧存引流售卖券
        
        抵扣类型：抵扣现金
        
        售价&抵扣金额：默认5元
        
        注意：售价需与抵扣金额保持一致
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/b3e5eff2-a89b-4cd5-b5e3-988f14f47d06.png)
        
    2.  **退款操作**
        
        此处为测试订单演示，优惠券名称内容以实际为准
        
        手动关闭订单及订单非异常退款时，优惠券不会同步退款
        
        需要手动点击优惠券内容边上的“退款”按钮进行退款
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/38deb8ca-6425-4dac-bf94-b37677039eb4.png)
        
        对优惠券退款可以手动选择是否撤销优惠券，撤销后优惠券失效
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/f3b68ae6-35a3-4b81-b077-52f64163d79e.png)
        
    3.  **优惠券订单页**
        
    
    进入优惠券订单页可以根据用户手机号、商户等快速搜索对应优惠券订单并进行退款及撤销操作（需要对应寄存订单结束）
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/e3784d75-373d-4bee-8d52-e9fd12ec3255.png)
    
    点击查看进入优惠券订单详情，能够在详情页中进行退款及撤销
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jGxzje5lvD/img/2b0c695e-3282-4cec-a9ea-2de6e37cde3a.png)

    2.  **用户下单后可能会出现的问题**
        
        Q：为什么我订单详情中有优惠券但是支付金额不变
        
        这个优惠券只是用于系统标记，并不产生实际优惠
        
    3.  **补充说明**
        

1.  以下情况出现退款时，优惠券会同步退款并撤销
    
    （1）异常原因10分钟内免费结束
    
    （2）预约免费时间内结束订单
    
    （3）收费规则免费时长内结束订单
    
    （4）下单失败
