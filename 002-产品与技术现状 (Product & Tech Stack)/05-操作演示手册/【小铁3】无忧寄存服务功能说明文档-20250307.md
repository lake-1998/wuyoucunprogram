# 【小铁3】无忧寄存服务功能说明文档

无忧寄存服务即为保险业务，目的是拓展公司营收，当前只在小铁3有部署；

1.  **后台管理系统操作指南**
    
    1.  **后台管理端**
        
        打开配置开关路径： PC管理端 -> 【商户配置】 -> 【业务配置】 -> 【寄存无忧服务】
        
        可以选择1元权益/2元权益或关闭功能，同时也支持一键把全部商户的保险业务关闭，需要找研发处理；

        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/3M0Oz58kjAdLqzeE/img/4e02648e-3d0d-4cbf-875b-66086d71fd45.png)
        
    2.  **角色权限隔离**
        
        在PC后台给平台及以下角色配置权限后，商户账号看不到PC后台的这些功能，只能看到有“寄存无忧服务订单”这个菜单，只有平台及以上账号配置好了权限，才可以看到这些功能；
        
        订单管理 -> 寄存无忧订单管理-退款：这个权限是控制列表中的退款按钮
        
        订单管理 -> 寄存无忧订单管理-列表：这个权限是控制列表展示，如果只勾选了退款按钮还是会展示列表
        
    3.  **管理端保险订单管理**
        
        订单一共有4个状态：【保障中】、【已结束】、【已退款】、【投保失败】，前2个状态对应寄存订单的【进行中】，【已结束】；保险订单退款状态是单独的与寄存订单退款与否无关，当出现投保失败的时候请与研发联系；
        
        支持退款、备注及导出操作；点击寄存订单编号可以直接跳转到对应的寄存订单详情页；

2.  **用户端业务流程说明**
    
    1.  **用户操作**
        
        当用户在开启了保险业务的网点进行正常的下单流程时，在下单页可以看到一个寄存无忧服务的复选框，用户如果不勾选这个复选框进行下单时会有一个挽留弹窗，如果勾选后，进行取消勾选操作也会有一个挽留弹窗；
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/3M0Oz58kjAdLqzeE/img/54d9efae-9afc-4562-b999-9ee5db0f8cdf.png)
        
        用户购买了保险下单后在首页订单卡片位置会有寄存无忧服务的标识，且点击进入订单详情页后也可以看到自己购买的寄存无忧服务的金额和业务详情，其中业务详情需要点击进入，权益内容中有第三方的联系方式；
        
        ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/3M0Oz58kjAdLqzeE/img/cbb31a10-e435-4ad0-8f99-4e89a5de991b.png)
        
    2.  **用户购买后可能会出现的问题**
        
        Q：一开始预付费是5块为什么我选择了购买保险，我付的钱还是5块？
        
        注意：这里做了隐性处理，不会让用户有感知，保费的金额会在预付的5块中扣除
        
        （这种情况有一个前提条件，就是网点的收费规则是按时收费且预付费大于保费金额才会出现）
        
        **预付＞基础收费时回复**：A:保费会从您的预付费用扣除哦，您可以在订单详情内查询相关的收费明细
        
        Q：为什么我在下单页显示20元付费，在支付的时候告诉我要支付21元/22元？
        
        注意：这里的网点收费规则如果是按柜的时候就会出现的情况，预付费20元+保费1元/2元=21元/22元
        
        **预付=基础收费时回复**：A:您这边刚刚有勾选了一个保险服务，这边保险服务是需要单独收费的哦，您下单后可以在订单详情内查询相关的收费明细
        
        Q：我该如何理赔？/对保险条款有疑问
        
        A：存包柜提供的是第三方保险服务，您可以拨打下保险服务4000785968进行咨询
        
    3.  **补充说明**
        

1.  目前商户端的营收统计里面不包含保险的金额，并且这个事情不用提前告知场地方，如果场地方有疑问找到客服，客服需要统一先联系陈星雨确认
    

2.商户端账号在【寄存柜订单】中看不到保险相关的信息，也没有【寄存无忧服务订单】列

3.当订单进行中时，对保险进行退款，将结束保险服务，失去标识，但是不影响订单进行

4.赔付包含2个部分：

*   用户正常寄存过程种的行李损坏（赔给用户）
    
*   用户寄存的物品导致柜子损坏（赔给小铁）
