# 小铁寄存柜美团上架流程说明

2.  团购券上线流程
    

1.  进入美团开店宝
    
2.  点击左侧“交易管理”-“团购”进入团单管理
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/acd73d11-f91d-42a3-9787-9f2aa9018ba4.png)
    
3.  点击新建团单
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/e59f33e0-c934-4396-9a3f-f734c84f9e3f.png)
    
4.  选择“迷你仓”团购分类（该分类可能后续发生更改，需关注），确认后进入下一步
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/984a88f2-3176-46b7-8f45-ca1c865f35b3.png)
    
5.  在团购信息中“适用门店”处，可以选择该团购券适用的门店，有两种上传方式，第一种是手动勾选现有门店，第二种上传对应表格（表格内门店必须是已在美团上线的）
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/9a00b8eb-dca0-408d-8321-a391edcfdb0b.png)
    
    ![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/b93a377e-3f2b-421d-a4ce-2e03187d3f8d.png)
    
    相关表格名称：小铁门店明细（4148家）更新码链接.xlsx
    
6.  团购券id与小铁后台id绑定，团购券创建成功后，生成团购id，将该id与“兑换id”对应关联。然后在右上角选择团购券所属网点（一次只能添加一个网点），如果需要批量添加则需要整理出表格提供给研发同事通过跑脚本一次性添加![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/ed64cc09-4d98-4b5f-8c0b-8fd2cff55046.png)
    

![image.png](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/res/J9LnW6jzAe73rlvD/img/cfd26e01-0ddc-42a1-8e58-9daf97d2c7f2.png)

2.  门店上线美团流程
    

1.  门店自动下架：目前在小铁2/小铁3停用对应网点时，美团上对应的网点也会下架
    
2.  新建门店：我方提供寄存网点信息给美团，美团统一上线我们的门店然后提供门店id给我们，通过研发绑定授权关系
