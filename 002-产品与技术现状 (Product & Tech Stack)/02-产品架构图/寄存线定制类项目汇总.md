### 功能框架

![image](https://alidocs.oss-cn-zhangjiakou.aliyuncs.com/a/pPbYG2LvMuy51lab/aca25ac4d7dc4ff59aef03d2b42b949e2624.png)

### 共性说明

小铁寄存柜产品能力在绝大多数项目中都有被复用场景

1.  存取包业务流程：纯用户端流程、纯终端流程、用户端结合终端流程
    
2.  鉴权方式
    
    | **鉴权方式** | **说明** |
    | --- | --- |
    | 密码取包 | 小程序支持使用手机号及密码存、取包，终端使用手机号及密码存包 |
    | 人脸取包 | 支持使用人脸拍照下单 |
    | 取件码取包 | 用户在终端使用特定取件码开柜取物 |
    | 刷身份证取包 | 终端刷身份证，生成二维码，手机扫码下单（首页是存取按钮，无二维码） |
    | 手环取包 | 仅支持终端使用手环取包 |
    | 腕带取包 | 仅支持终端使用腕带取包 |
    | 掌纹存包 | 掌纹识别模块 |
    | 终端身份证存包 | 终端全流程刷身份证下单取包（该模式支持与密码取包同时开启） |
    | 护照/港澳通行证鉴权 | 待定项目··· |
    
3.  管理后台能力：多层级组织管理、多订单类型管理、IOT管理、运营配置、异常监控、数据统计、财务管理···
